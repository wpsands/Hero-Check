# Brand Positioning Document

## Company
Velocitab — The API Platform for Product-Led Growth

## Category
API Infrastructure & Developer Tools

## Target Buyer
CTOs and VP Engineering at B2B SaaS companies building API-first products (Series A through Series C)

## Core Positioning Statement
Velocitab gives engineering teams a single platform to design, deploy, and monetize APIs — so they can ship integrations in hours instead of sprints and turn their API into a revenue channel.

## Key Messages

1. **Ship Integrations Faster**: Design, mock, and deploy production APIs from a single workspace. Built-in versioning, rate limiting, and SDK generation eliminate weeks of boilerplate.
2. **Monetize Your API**: Usage metering, tiered pricing, and developer billing built in. Launch a paid API program without building billing infrastructure from scratch.
3. **Developer Experience as a Feature**: Auto-generated docs, interactive playground, and OAuth flows that work out of the box. Your API becomes the product, not a side project.

## Differentiators
- Unified: Design, deploy, meter, and bill from one platform (competitors require 3-4 tools)
- API-native billing: Usage-based pricing engine purpose-built for API products, not retrofitted from SaaS billing
- SDK generation: Auto-generate client libraries in 8 languages from your OpenAPI spec
- Self-serve onboarding: Developer portal with instant API keys, sandbox environments, and interactive docs

## Proof Points
- 2,000+ APIs managed across 400+ companies
- 99.99% gateway uptime over trailing 12 months
- Average integration time reduced from 3 weeks to 2 days
- $18M Series B led by tier-1 enterprise VC

## Competitive Positioning
- vs. API Gateways (Kong, Apigee): Velocitab adds monetization and developer portal; gateways are infrastructure-only
- vs. Billing platforms (Stripe Billing, Orb): Velocitab is API-aware with metering at the request level, not invoice-level
- vs. Doc tools (ReadMe, Mintlify): Velocitab generates docs as part of the full lifecycle, not as a standalone layer
